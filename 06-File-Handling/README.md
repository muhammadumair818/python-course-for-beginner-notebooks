# 06-File-Handling: Working with Files

Learn to read, write, and handle files safely.

## Topics Covered

### 1. **File I/O** (`01_file_io.py`)
- Reading and writing text files
- Working with file modes

### 2. **CSV & JSON** (`02_csv_json.py`)
- Reading/writing CSV files
- Working with JSON data

### 3. **Exceptions** (`03_exceptions.py`)
- Try-except blocks
- Handling errors gracefully
- Common exception types

## File Modes

| Mode | Purpose |
|------|---------|
| 'r' | Read |
| 'w' | Write |
| 'a' | Append |
| 'x' | Create |
