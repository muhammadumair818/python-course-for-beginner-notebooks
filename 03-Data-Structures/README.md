# 03-Data-Structures: Collections & Storage

Master different ways to organize and store data in Python.

## Topics Covered

### 1. **Lists** (`01_lists.py`)
- Creating and indexing
- Slicing and modifying
- Common list methods
- Looping and comprehension

### 2. **Tuples** (`02_tuples.py`)
- Creating immutable sequences
- Unpacking values
- Using as dictionary keys

### 3. **Sets** (`03_sets.py`)
- Unique item collections
- Set operations (union, intersection, difference)
- Removing duplicates

### 4. **Dictionaries** (`04_dictionaries.py`)
- Key-value pairs
- Accessing and modifying
- Looping through dictionaries

### 5. **Exercises** (`05_exercises.py`)
- Comprehensive practice problems

## Quick Reference

| Structure | Ordered | Mutable | Duplicates |
|-----------|---------|---------|-----------|
| List | Yes | Yes | Yes |
| Tuple | Yes | No | Yes |
| Set | No | Yes | No |
| Dict | No | Yes | No |
