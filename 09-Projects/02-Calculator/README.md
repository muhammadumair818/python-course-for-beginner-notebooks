# Calculator

A simple command-line calculator for basic arithmetic operations.

## Features

- Add, Subtract, Multiply, Divide
- Modulo and Power operations
- Calculation history (last 5)
- Clear history
- Error handling

## How to Run

```bash
python calculator.py
```

## Menu Options

1. **Add** - Sum two numbers
2. **Subtract** - Difference of two numbers
3. **Multiply** - Product of two numbers
4. **Divide** - Quotient of two numbers
5. **Modulo** - Remainder of division
6. **Power** - Raise to power
7. **Clear History** - Delete calculation history
8. **Exit** - Quit the program

