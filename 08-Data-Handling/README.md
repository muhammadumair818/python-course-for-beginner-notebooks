# 08-Data-Handling: Advanced Data Analysis

Learn pandas, numpy, and visualization libraries.

## Content

### Jupyter Notebooks

1. **01_pandas_dataframe.ipynb** - DataFrame operations, filtering, grouping
2. **02_numpy_arrays.ipynb** - Array operations and numerical computing
3. **03_matplotlib_seaborn.ipynb** - Data visualization with plots

## Key Topics

- **Pandas**: DataFrames, Series, merging, grouping, aggregation
- **NumPy**: Arrays, operations, broadcasting
- **Matplotlib/Seaborn**: Line plots, bar charts, heatmaps, scatter plots

## Running Jupyter

```bash
jupyter notebook 01_pandas_dataframe.ipynb
```
