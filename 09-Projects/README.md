# 09-Projects: Real-World Python Applications

Six complete projects to build your portfolio and strengthen your skills.

## Projects Included

### 1. **Number Guessing Game** (`01-Number-Guessing-Game/`)
A fun interactive game where the computer picks a number and you guess it.

**Key Concepts:**
- Random number generation
- While loops
- User input validation
- Conditional logic

**Run:** `python game.py`

---

### 2. **Calculator** (`02-Calculator/`)
Performs basic arithmetic operations with history tracking.

**Key Concepts:**
- Function definitions
- Menu-driven interfaces
- Input validation
- Error handling

**Run:** `python calculator.py`

---

### 3. **Todo List App** (`03-Todo-List/`)
Task management system with persistent file storage.

**Key Concepts:**
- Class-based design
- File I/O (JSON)
- Data persistence
- CRUD operations

**Run:** `python todo.py`

---

### 4. **Web Scraper** (`04-Web-Scraper/`)
Extract data from websites using BeautifulSoup.

**Key Concepts:**
- HTTP requests
- HTML parsing
- DOM traversal
- CSV file writing
- **Note:** Ethics and legal considerations

**Run:** `python scraper.py`

**Installation:**
```bash
pip install requests beautifulsoup4
```

---

### 5. **Data Analysis** (`05-Data-Analysis/`)
Analyze datasets and create visualizations.

**Key Concepts:**
- Pandas DataFrames
- Data cleaning
- Exploratory analysis
- Matplotlib/Seaborn plots

**See:** README.md in folder

---

### 6. **Password Strength Checker** (`06-Password-Strength-Checker/`)
Evaluate password security with detailed feedback.

**Key Concepts:**
- Regular expressions (regex)
- String validation
- Pattern matching
- Security best practices

**Run:** `python checker.py`

---

## How to Use These Projects

### For Learning:
1. **Read** the README in each project folder
2. **Study** the source code comments
3. **Run** the program and test it
4. **Modify** the code to add features
5. **Expand** with your own ideas

### For Your Portfolio:
1. Clone/download each project
2. Test thoroughly
3. Add your own features or improvements
4. Create a GitHub repository
5. Write comprehensive README files
6. Deploy or showcase on your portfolio

### Progressive Difficulty

1. **Beginner:** Number Guessing Game, Calculator
2. **Intermediate:** Todo List, Password Checker
3. **Advanced:** Web Scraper, Data Analysis

## Installation & Setup

### Basic Requirements

```bash
python 3.7+
```

```bash

```
